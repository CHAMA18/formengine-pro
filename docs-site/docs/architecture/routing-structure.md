# Routing Structure
User-facing pages: /, /signin, /signup, /dashboard, /forms/new, /templates, /submissions, /api-keys, /settings, /docs/api, /f/[shareId]. Internal API: /api/*. Public REST API: /api/v1/* (API key auth required).
