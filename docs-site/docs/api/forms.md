# Forms API

- **GET /api/v1/forms** — List published forms (scope: `forms:read`)
- **POST /api/v1/forms** — Create form from flowchart (scope: `forms:write`)
- **GET /api/v1/forms/\{shareId\}** — Get form schema (scope: `forms:read`)
