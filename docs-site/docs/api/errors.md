# Error Codes
| Code | Meaning |
|------|---------|
| 400 | Bad request (missing fields) |
| 401 | Unauthorized (missing/invalid API key) |
| 403 | Forbidden (insufficient permissions) |
| 404 | Not found |
| 422 | Validation failed (returns field-level errors) |
| 500 | Server error |
