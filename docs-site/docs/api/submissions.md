# Submissions API
- POST /api/v1/forms/{shareId}/submissions — Submit response (scope: submissions:write)
- GET /api/v1/forms/{shareId}/submissions — List submissions (scope: submissions:read)
- GET /api/v1/submissions — List all submissions (scope: submissions:read)
