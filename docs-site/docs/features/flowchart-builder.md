# Flowchart Builder
Infinite pannable/zoomable canvas. Drag nodes from palette, connect with drag-to-connect handles, configure in inspector. Real-time JSON schema preview. Publish with one click. Keyboard shortcuts: Escape (cancel), Delete (remove node).
